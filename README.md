# AnnAamrit | भारत की रसोई 🍛

**Premium Home-Style Meals & Lunch Box Delivery Service in Patna.**

This repository contains the official frontend codebase for the AnnAamrit website. Built with a vision to scale to a 1000Cr+ food ecosystem, the architecture prioritizes performance, premium aesthetics, and user trust. 

## 🚀 Key Features

* **Premium UI/UX:** Modern glassmorphism navigation, smooth scroll-reveal animations, and a rich brand palette (Deep Green, Gold, Saffron).
* **Dynamic Menu Engine:** Weekly menu populated via JavaScript, ready for API integration.
* **Seamless Lead Generation:** Custom subscription form designed to capture user data (Students, Office Workers, Families) securely.
* **Instant Action:** Direct WhatsApp ordering integration for high-conversion sales.
* **Blazing Fast:** Zero-dependency vanilla HTML/CSS/JS architecture for 100/100 Lighthouse performance scores.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, Vanilla JavaScript (No heavy frameworks, ensuring max speed)
* **Fonts:** Playfair Display (Serif) & Poppins (Sans-serif)
* **Planned Hosting:** Cloudflare Pages / Vercel (Global CDN)
* **Planned Backend:** Google Sheets API / Vercel Serverless Functions
